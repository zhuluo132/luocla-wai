# LuoCla Wai — 朱阳 AI 云端版

> 手机浏览器直接访问，无需安装，无需电脑开机。

## 在线使用

👉 **[https://zhuluo132.github.io/luocla-wai/](https://zhuluo132.github.io/luocla-wai/)**

## 功能

- 🤖 DeepSeek V4 双模型（flash / pro）
- 🌐 联网搜索（Wikipedia）
- 📚 20+ AI 技能面板
- 👥 多用户切换（独立记忆）
- 🌙 6套主题（暗黑/明亮/Ocean/Dracula/Sunset）
- 📱 手机端优化（PWA，可添加到主屏幕）

## 首次使用

1. 打开上方链接
2. 点击设置图标 ⚙️
3. 输入你的 DeepSeek API Key（[点击获取](https://platform.deepseek.com/api_keys)）
4. 开始对话！

## 本地部署

```bash
git clone https://github.com/zhuluo132/luocla-wai.git
# 直接用浏览器打开 index.html 即可
```
